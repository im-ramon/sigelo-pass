#### 🧭 <a id="requirements">Pré-requisitos para utilizar a aplicação</a>:

  - Ter o perfil de acesso previamente autorizado; e
  - Possuir smartphone com:
    - Sistema Operacional Android.
    - Câmera.
    - Acesso à internet.
